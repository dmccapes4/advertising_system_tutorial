2 · LLM = heuristic infrastructure

Regex and if/elif trees collapse into a sentence in the system prompt. You own intent and contracts; the model handles the brittle matching.

Before — brittle code
if re.match(r"refund|return", q):
    route = "refunds"
elif re.match(r"ship|track", q):
    route = "shipping"
elif "fair use" in q.lower():
    route = "legal"
else:
    route = "general"  # … grows forever
→
After — one prompt line

"Route the user question to refunds, shipping, legal, or general — use the advertising knowledge tool for legal/fair-use questions and cite book + page."
